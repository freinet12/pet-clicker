# pet-clicker

## Project setup
Create a .env file and add the following variable:
`VUE_APP_PET_CLICKER_API_URL="{{docker_app_url}}`
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
